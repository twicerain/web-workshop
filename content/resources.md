## Glossary

Concise expansions on some abbreviations or domain specific terms as this was mostly intended to pad out the site to demo the router and navigation.

API
: Application Programming Interface

CDN
: Content Delivery Network

CLI
: Command Line Interface

CMS
: Content Management System

CSR
: Client Side Rendered

DOM
: Document Object Model

ESM
: ECMAScript Modules

HTML
: HyperText Markup Language

HTTP
: Hypertext Transfer Protocol

Hydration
: Process where the browser executes JavaScript to make a static HTML page interactive

Islands
: Architecture where only interactive components are hydrated, leaving the rest as static HTML

ISR
: Incremental Static Regeneration

Jamstack
: JavaScript, APIs, and Markup - modern architecture using client-side JS, reusable APIs, and prebuilt markup

JS
: JavaScript

JSON
: JavaScript Object Notation

JSX
: JavaScript XML

LSP
: Language Server Protocol

MPA
: Multi Page Application

MVC
: Model View Controller

REPL
: Read-Eval-Print Loop

SEO
: Search Engine Optimization

SPA
: Single Page Application

SSG
: Static Site Generator

SSR
: Server Side Rendered

UI
: User Interface

URL
: Uniform Resource Locator
